Day 4 - Oceania/Australian Relay - Cootamumdra

Today we woke up at 6:00 so it was a nice change from the really early starts so we got a full breakfast from the hotel. Mr Purcell, Angus and I arrived a few minutes before everyone else so we set up the LED Screen and took off the protection from the finish. Because we were at the same location two days in a row the infrastructure was pretty much all setup so we were ready to go. Things were pretty hectic at the finish tent because the relays meant that the runners' data had to be sorted in a correct way and the orienteers changed their teams without telling us so that caused a lot of problems which I had to fix.
